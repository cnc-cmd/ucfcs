生活技能:
1. 驾驶证:强推，一生必用
2. 保安证:浙江温州40元，当场领证
3. 急救证:学校统一考出

专业技能:
1. 全国计算机等级考试（NCRE）:毕业需要(建议浙江省计算机等级考试，只要30元)
